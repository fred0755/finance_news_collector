
阶段	模块	任务编号	任务描述	核心产出与验收标准
第一阶段	1. 采集模块	T1.1	改造或新建采集器：在 src/collectors/ 下创建 eastmoney_collector.py，实现从东方财富快讯(kuaixun.eastmoney.com)稳定抓取新闻列表（标题、链接、时间）。	能运行并打印出至少5条正确的新闻条目。
T1.2	实现数据存储：设计 src/storage/models.py 中的 NewsArticle 数据模型，并编写 database.py 完成数据库（SQLite或MySQL）初始化与连接。	执行脚本后，数据库内成功创建 news_articles 表。
T1.3	集成采集与存储：修改采集器，使其抓取到的数据能通过storage模块存入数据库。	抓取的新闻能带着基础字段（标题、链接、时间）存入数据库。
2. 分析模块	T1.4	实现基础去重：在 src/processors/deduplicator.py 中实现基于URL哈希的第一级去重逻辑。	相同的新闻链接不会在数据库中重复出现。
T1.5	实现重要性评分(v0.1)：在 src/processors/analyzer.py 中实现一个规则引擎，根据来源和标题关键词给新闻打0-10分。	存入数据库的每条新闻都有一个 importance_score 字段。
3. 推送模块	T1.6	集成企业微信推送：配置 src/config/settings.py 中的微信密钥，在 src/push/wechat_service.py 中实现推送函数。	能手动调用函数，向自己发送一条测试消息。
4. 系统集成	T1.7	编写调度主入口：创建 main.py，串联采集、分析、推送流程，形成可手动执行的完整流水线。	执行 python main.py 能完成一次从采集到推送的全流程测试。
